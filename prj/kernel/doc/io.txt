For the introduction of user model, refer to user.txt.

- (succ)IO_SET_CALLBACK(handler_func)

Set the userspace entry of I/O callback. Returns 0 with success.

- (succ)IO_SET_TLS(io_buf_size, user_size)

Realloc the TLS storage, must be called with no I/O in buffer. The
user storage would be copy if successed. Returns 0 with success.

- (succ)IO_CREATE_PROC(bin, bin_size, flags)

Create a new process(thread) with specified binary image. Returns 0
with success.

- IO_EXIT()

Exit current process(thread)

- (succ)IO_BRK(end)

Reset the user end address

- (succ)IO_PAGE_HOLE_SET(base, size)

- (succ)IO_PAGE_HOLE_CLEAR(base, size)

- (succ, base, size) | IO_PAGE_HOLE_GUARD()

- (succ)IO_DEBUG_PUTCHAR(ch)

putchar on debug console

- (succ)IO_DEBUG_GETCHAR()

getchar on debug console

- (nic)IO_NIC_OPEN()

alloc a new NIC

- (succ)IO_NIC CLOSE(nic)

free a NIC

- (succ, buf, req_id, buf_size) | IO_NIC_NEXT_REQ_R(nic, ack_req_id, ack_size)

acknowledge previous req and wait for a read request on specific NIC

- (succ, buf, req_id, buf_size) | IO_NIC_NEXT_REQ_W(nic, ack_req_id, ack_size)

acknowledge previous req and wait for a write request on specific NIC
